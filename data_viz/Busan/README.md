**"영역별 지표에 따른 부산시 불균형 실태 확인" - 구/군 기준**

- 인구활력 : 인구증감률, 고령인구비율(-), 합계출산율<br>
- 산업 · 경제 : 재정자립도, 사업체 종사자 수 , 기초생활 수급자 수 비율(-)<br>
- 주거생활환경 : 노후주택률(-), 지하철역사 수, 자동차등록대수당 주차장 면수<br>
- 교육문화복지 : 초등학교 수, 문화기관 수, 의료기관 수<br>

=> 4가지 영역에서 3가지 지표를 선정하여 점수 부여
